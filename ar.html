Arduino Code

The ESP32 firmware for the LadyBug robot — combining Wi-Fi web control and autonomous IR-based obstacle avoidance in a single sketch.

Code Overview Full Code

Code Highlights


• Platform: ESP32 (Wi-Fi)

• Driver: L298N (ENA/ENB + IN1–IN4)

• Sensor: IR obstacle sensor (Pin 4)

• Modes: Auto & Manual

• Web UI: HTTP route handlers

Pin Definitions

ESP32 GPIO assignments for the L298N motor driver and IR sensor.

IR_PIN
GPIO 4

IR obstacle sensor input. Reads LOW when an obstacle is detected in front of the robot.

ENA
GPIO 14

Enable pin for the left motor channel of the L298N. Set HIGH to enable PWM drive.

IN1 / IN2
GPIO 27 / 26

Direction control for the left motor. IN1 HIGH + IN2 LOW = forward; reversed = backward.

ENB
GPIO 25

Enable pin for the right motor channel of the L298N. Set HIGH to enable drive.

IN3 / IN4
GPIO 33 / 32

Direction control for the right motor. IN3 HIGH + IN4 LOW = forward; reversed = backward.

Code Structure

How the sketch is organised across its key functional sections.

Wi-Fi & Web Server Setup

The ESP32 connects to a Wi-Fi network using the provided SSID and password, then starts an HTTP server on port 80. The IP address is printed to Serial so the user can navigate to the robot's control page from any browser on the same network.

Motor Control Functions

Four direction functions — forward(), backward(), left(), right() — and a stopMotor() function set the IN1–IN4 pin states on the L298N to drive the robot in each direction. These are called by both the web handlers and the auto-mode logic.

Dual Mode System

The robot operates in two modes controlled by an enum: AUTO_MODE (default) and MANUAL_MODE. Switching to manual stops the motors and hands control entirely to the web interface. Switching back to auto resumes the obstacle avoidance loop immediately.

HTTP Route Handlers

Seven routes are registered: /, /auto, /manual, /forward, /backward, /left, /right, and /stop. Each handler updates the mode or calls a motor function and returns the control webpage, keeping the browser in sync with the robot's state.

Auto Mode Logic

In AUTO_MODE the main loop reads the IR sensor every iteration. If the path is clear, the robot moves forward. If an obstacle is detected, it stops, waits 2 seconds, then rotates right in a tight loop — calling server.handleClient() inside the loop to keep Wi-Fi responsive — until the path clears, then resumes forward movement.

Functional Sections

A quick reference for each block of the sketch.

Libraries

WiFi & WebServer

Includes WiFi.h and WebServer.h from the ESP32 Arduino core to enable Wi-Fi connectivity and HTTP server functionality.

Web Page

Inline HTML UI

A minimal HTML control page served as a string — contains mode-switch buttons (AUTO / MANUAL) and directional controls (FORWARD, BACKWARD, LEFT, RIGHT, STOP).

setup()

Initialisation

Sets all motor pins as OUTPUT, enables ENA and ENB HIGH, connects to Wi-Fi, prints the IP address, registers all route handlers, and starts the web server.

loop()

Main Loop

Calls server.handleClient() on every iteration to process incoming HTTP requests, then runs autoMode() if the current mode is AUTO_MODE.

const char* password = "lamb0710"; WebServer server(80); /******** IR SENSOR ********/ #define IR_PIN 4 /******** L298N ********/ #define ENA 14 #define IN1 27 #define IN2 26 #define ENB 25 #define IN3 33 #define IN4 32 /******** MODES ********/ enum Mode { AUTO_MODE, MANUAL_MODE }; Mode currentMode = AUTO_MODE; /******** MOTOR FUNCTIONS ********/ void forward() { digitalWrite(IN1, HIGH); digitalWrite(IN2, LOW); digitalWrite(IN3, HIGH); digitalWrite(IN4, LOW); } void backward() { digitalWrite(IN1, LOW); digitalWrite(IN2, HIGH); digitalWrite(IN3, LOW); digitalWrite(IN4, HIGH); } void left() { digitalWrite(IN1, LOW); digitalWrite(IN2, HIGH); digitalWrite(IN3, HIGH); digitalWrite(IN4, LOW); } void right() { digitalWrite(IN1, HIGH); digitalWrite(IN2, LOW); digitalWrite(IN3, LOW); digitalWrite(IN4, HIGH); } void stopMotor() { digitalWrite(IN1, LOW); digitalWrite(IN2, LOW); digitalWrite(IN3, LOW); digitalWrite(IN4, LOW); } /******** WEB PAGE ********/ String webpage = "<html><head>" "<meta name='viewport' content='width=device-width, initial-scale=1'>" "</head><body>" "<h1>ESP32 Robot</h1>" "<h2>Mode</h2>" "<a href='/auto'><button>AUTO MODE</button></a>" "<a href='/manual'><button>MANUAL MODE</button></a>" "<br><br>" "<a href='/forward'><button>FORWARD</button></a>" "<a href='/backward'><button>BACKWARD</button></a>" "<br><br>" "<a href='/left'><button>LEFT</button></a>" "<a href='/right'><button>RIGHT</button></a>" "<br><br>" "<a href='/stop'><button>STOP</button></a>" "</body></html>"; /******** HANDLERS ********/ void handleRoot() { server.send(200, "text/html", webpage); } void handleAuto() { currentMode = AUTO_MODE; server.send(200, "text/html", webpage); } void handleManual() { currentMode = MANUAL_MODE; stopMotor(); server.send(200, "text/html", webpage); } void handleForward() { if(currentMode == MANUAL_MODE) forward(); server.send(200, "text/html", webpage); } void handleBackward() { if(currentMode == MANUAL_MODE) backward(); server.send(200, "text/html", webpage); } void handleLeft() { if(currentMode == MANUAL_MODE) left(); server.send(200, "text/html", webpage); } void handleRight() { if(currentMode == MANUAL_MODE) right(); server.send(200, "text/html", webpage); } void handleStop() { stopMotor(); server.send(200, "text/html", webpage); } /******** AUTO MODE ********/ void autoMode() { int obstacle = digitalRead(IR_PIN); if(obstacle == LOW) { stopMotor(); delay(2000); // turn right until path is clear while(digitalRead(IR_PIN) == LOW) { right(); delay(100); server.handleClient(); } stopMotor(); delay(100); forward(); } else { forward(); } } /******** SETUP ********/ void setup() { Serial.begin(115200); pinMode(IR_PIN, INPUT); pinMode(ENA, OUTPUT); pinMode(IN1, OUTPUT); pinMode(IN2, OUTPUT); pinMode(ENB, OUTPUT); pinMode(IN3, OUTPUT); pinMode(IN4, OUTPUT); digitalWrite(ENA, HIGH); digitalWrite(ENB, HIGH); stopMotor(); WiFi.begin(ssid, password); Serial.print("Connecting"); while(WiFi.status() != WL_CONNECTED) { delay(500); Serial.print("."); } Serial.println(); Serial.println("Connected"); Serial.print("IP Address: "); Serial.println(WiFi.localIP()); server.on("/", handleRoot); server.on("/auto", handleAuto); server.on("/manual", handleManual); server.on("/forward", handleForward); server.on("/backward", handleBackward); server.on("/left", handleLeft); server.on("/right", handleRight); server.on("/stop", handleStop); server.begin(); } /******** LOOP ********/ void loop() { server.handleClient(); if(currentMode == AUTO_MODE) { autoMode(); } }
