Activities

Progressive hands-on lab sessions in Platform Based Development.

00

Introduction

Overview of platform-based development and setting up the Arduino IDE with ESP32 board support.

01

LED, Buzzer & LDR Sensor

Understanding circuit connections and programming the ESP32 to control an LED and buzzer with LDR-based light sensing.

02

Stepper Motor & Relay

Wiring and programming a stepper motor with a relay module to control loads via the ESP32 digital outputs.

03

Advanced Circuit Integration

Combining multiple components — motors, sensors, and relays — into a single integrated circuit system.

04

Servo Motor Wiring

Soldering and wiring a servo motor, learning how pulse-width modulation controls its angular position.

05

Final System Integration

Bringing all components together into a complete working system — a full demonstration of platform-based development.

Learning Journey

Setup

Installed Arduino IDE and configured the ESP32 Dev Module board. Uploaded the first sketch to verify the connection and environment.

First Blink

Wrote and uploaded a basic LED blink program — the classic first step in embedded programming — to confirm digital output control.

Sensor Logic

Connected an LDR sensor and buzzer, writing conditional code to trigger an audio alert based on detected ambient light levels.

Actuator Control

Programmed a stepper motor with a relay module, understanding how digital signals drive physical rotational and switching movement.

Full Integration

Assembled and demonstrated a complete multi-component system with servo, LDR sensor, buzzer, and relay all working in unison.

Featured Work

LED & LDR Response Circuit

An automatic light-sensing circuit where the LDR triggers a buzzer when ambient light drops below a set threshold.

Stepper Motor with Relay Control

A relay-switched stepper motor circuit programmed to run in forward and reverse using ESP32 digital outputs.

Integrated Multi-Component System

A final-stage project combining LED, buzzer, stepper motor, relay, and servo into a fully functional embedded system.
